# Max Wroblewsky Website

Personal resume and portfolio site for [max.wroblewsky.com](https://max.wroblewsky.com).

## Structure

- `index.html`: main portfolio page
- `style.css`: site styling and animation
- `images/`: project screenshots used in the project pop-outs
- `assets/headshots/`: profile and side headshots
- `assets/docs/`: resume PDF
- `privacy.html` and `terms.html`: supporting legal pages

## Local Preview

Because this is a static site, you can preview it by opening `index.html` directly in a browser or by serving the folder with a lightweight local server.

## Notes

- Project groups stay as dropdown sections.
- Individual projects open in modal pop-outs.
- Asset references are organized under `assets/` instead of the repo root.
